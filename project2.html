Randomly Generating Moving Circles on a Screen

This was a quick project that I made integrating the knowledge of HTML, CSS, and JavaScript. This project generates random circles in a specific part of the screen. These circles move in randomly generated directions and never stop. It's something that I have found mesmerizing. This project introduced me to creating computer animations.





A Rotating Cube

This project mostly focuses on JavaScript rather than HTML and CSS. This cube calculates the translation and rotation of the vertices of the cube. It stores these coordinates in matrices. Any change in the coordinate plane is stored and is shown on screen.
